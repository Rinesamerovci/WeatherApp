Weather App 
React Native Weather App me current weather dhe 5-day forecast duke përdorur OpenWeather API. App-i ka dizajn modern me ngjyra pistachio dhe cards profesionale.

Features
Shfaq motin aktual për qytetin e zgjedhur
Temperature
Description
Humidity
Wind speed
5-Day Forecast me ikonat e motit
Input për të ndryshuar qytetin
Async/await dhe error handling me try/catch
Dizajn modern dhe responsive për mobile
